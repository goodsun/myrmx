## カラースキーム

- **背景**: ダークモード対応（白/黒）
- **プライマリ**: #3B82F6 (青)
- **成功**: #10B981 (緑)
- **エラー**: #EF4444 (赤)
- **警告**: #F59E0B (黄)

## レスポンシブ対応

- モバイル: 縦スクロール、単一カラムレイアウト
- タブレット: 2カラムレイアウト
- デスクトップ: フル機能表示

## インタラクション

1. **プロジェクト選択時**
   - 自動的にコントラクト一覧を更新
   - コンパイル状態をチェック

2. **コンパイルボタン**
   - プログレス表示
   - エラー時は詳細を表示

3. **デプロイ時**
   - MetaMaskポップアップ
   - トランザクション待機中の表示
   - 成功/失敗の通知

4. **ABI表示**
   - 折りたたみ可能
   - シンタックスハイライト
   - ワンクリックコピー
